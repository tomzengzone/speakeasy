# Release Checklist

## Before Release
- [ ] MVP scope updated.
- [ ] Feature specs updated.
- [ ] Acceptance criteria mapped to tests.
- [ ] API contract updated.
- [ ] Domain schema updated.
- [ ] AI output schemas validated.
- [ ] All required tests pass.
- [ ] Implementation report updated.
- [ ] Quality report updated.
- [ ] Version log updated.
- [ ] Rollback plan reviewed.

## Production Controls
- [ ] Provider secrets are not bundled in client.
- [ ] Runtime configuration uses release-safe values.
- [ ] Error logging avoids sensitive payloads.
- [ ] Payment and auth settings are production-ready if enabled.
