# 用户故事

## 启动、登录与首评
- 作为未登录学习者，我希望启动后先看到登录页，以便进入自己的学习数据。
- 作为学习者，我希望在登录前同意服务条款和隐私政策，以便确认基础使用约束。
- 作为学习者，我希望可以选择微信、Apple、手机号验证码或邮箱入口登录，以便使用当前设备和账号条件下可用的方式进入 App。
- 作为已登录新用户，我希望完成首评，以便生成第一条学习路线。
- 作为学习者，我希望选择目标方向、表达卡点、当前输出水平和每日分钟数，以便首页能保存当前偏好和官方场景路线。
- 作为选择日常服务方向的学习者，我希望仍能完成首评进入首页，但不被误导为已有日常服务完整场景可练。

## 情景学习
- 作为学习者，我希望在首页看到情景学习、推荐表达和我的三个主入口，以便清楚进入当前 MVP 的核心流程。
- 作为学习者，我希望看到英语面试和入职介绍两个官方场景，以便选择真实可练的职场场景。
- 作为学习者，我希望查看场景简介、标签、目标等级、表达数量和当前进度，以便判断是否加入学习。
- 作为学习者，我希望搜索、筛选、加入、移除、设为当前场景并切换目标等级，以便维护自己的学习路线。
- 作为学习者，我希望首页优先提示未完成会话、到期复习、薄弱表达或下一条未掌握表达，以便直接继续最该做的练习。

## 听力热身与推荐表达
- 作为学习者，我希望先播放完整场景对话，以便熟悉语境和回答节奏。
- 作为学习者，我希望能上一句/下一句切换、暂停和循环播放，以便按自己的节奏听。
- 作为学习者，我希望切换到跟读模式录制候选人台词，以便获得完整度或发音反馈。
- 作为已加入场景的学习者，我希望在推荐表达页看到每日表达队列，以便集中处理复习、薄弱和表达变体。
- 作为学习者，我希望完成选择题、填空、意图回忆、接下句、替换槽位、变体改写、流利挑战或跟读等小任务，以便逐步把表达练成自己的话。
- 作为学习者，我希望收藏或取消收藏表达，以便在收藏页复看真正有用的表达。

## 语音模拟与教练反馈
- 作为学习者，我希望按场景和目标等级进入语音模拟，以便练习真实口语输出。
- 作为学习者，我希望查看当前目标进度和场景导航，以便知道本轮正在练哪个表达。
- 作为学习者，我希望卡住时请求提示，以便继续完成当前问题。
- 作为学习者，我希望录音后自动转写并提交回答，以便获得教练反馈。
- 作为学习者，我希望回答后看到教练反馈、重试建议、表达建议或下一问题，以便知道下一步怎么改。
- 作为学习者，我希望能播放或翻译教练消息，并播放自己的语音回答，以便复查听说效果。
- 作为中途退出的学习者，我希望下次进入同一场景同一等级时恢复未完成会话，以便不用从头开始。

## 复盘、复习与个人结果
- 作为学习者，我希望一轮练习结束后看到本轮掌握、总进度、遗忘曲线、薄弱标签和下轮重点，以便判断练习结果。
- 作为学习者，我希望掌握表达、薄弱表达、复习状态和个人素材影响后续首页或推荐表达，以便后续练习更聚焦。
- 作为学习者，我希望在收藏页看到去重后的收藏表达，以便复看和删除不需要的收藏。
- 作为学习者，我希望在个人中心看到学习概览、收藏摘要、技能分布、学习历史、学习报告入口和已完成场景入口，以便回顾长期学习结果。

## 我的与账号设置
- 作为学习者，我希望编辑头像和昵称，以便维护个人资料。
- 作为学习者，我希望设置每日提醒时间和开关，以便安排练习提醒。
- 作为学习者，我希望切换浅色/深色主题，以便适配使用环境。
- 作为学习者，我希望查看会员方案、发起购买或恢复购买，以便了解当前订阅入口。
- 作为学习者，我希望查看隐私政策和服务条款，以便了解数据和使用规则。
- 作为学习者，我希望可以退出登录或注销账号，以便管理账号状态。
