## Product Base 规则
- Product Base 是活的总需求库，承载已接受稳定行为。
- Increment 需求在完成实现、验收、追溯、测试和报告前，不得合并进 Product Base。
- Baseline 是从 Product Base 冻结出的快照，不得替代本文作为持续需求源。
- 本文只写用户目标、用户路径、功能需求、成功标准、非目标和边界；不写 API schema、prompt schema、UI 布局或代码实现。

## 稳定能力范围
当前 Product Base 覆盖一个已实现的官方职场场景学习闭环：学习者完成登录门禁和首评后，从首页进入英语面试或入职介绍官方场景，完成听力热身、推荐表达练习、语音场景模拟、教练反馈、表达收藏、复习沉淀和练习总结，并在首页、推荐表达、收藏页、个人 Wiki 或个人中心看到学习结果。

## 假设
- 当前稳定能力只承认可从现有前端入口进入、被用户观察到，并有代码或资产支撑的行为。
- 当前真实官方场景只包括 `job_interview` 和 `onboarding_introduction`。
- AI、ASR、TTS、评分、学习报告和登录成功依赖外部服务或平台配置；Product Base 只承认前端调用、状态展示、错误处理和可降级体验。
- 首评每日分钟数是偏好，不控制当前推荐表达队列长度。
- 收藏表达用于复看，不等同于生成复习任务。
- 会员页权益文案不等同于完整权益 gating 已实现。
- 离线内容和成就是占位页，不是完整业务闭环。

## 用户路径
1. 学习者打开 App，看到启动加载状态；启动失败时看到错误和重试入口。
2. 未登录学习者进入登录页；已登录但未首评进入首评；已完成首评进入首页。
3. 学习者在首评中选择目标方向、表达卡点、输出水平和每日分钟数。
4. 首评完成后，支持的官方场景选择写入学习路线；不支持的方向不写成真实场景。
5. 学习者在情景学习页搜索、筛选、加入、移除、设为当前场景或切换等级。
6. 学习者从首页进入听力热身，播放场景对话或进行跟读录音。
7. 学习者进入推荐表达，完成复习、补弱或表达变体任务，并收藏有用表达。
8. 学习者进入语音场景模拟，请求提示、录音作答、查看教练反馈、播放或翻译消息、查看场景导航。
9. 学习者中途退出后，再次进入同一场景同一等级时可恢复未完成会话。
10. 学习者完成一轮练习后查看总结，并让掌握、薄弱、复习和个人素材沉淀到后续页面。
11. 学习者在我的页面查看学习结果、收藏、历史、报告入口、会员入口和账号设置。

## 功能需求

### FR-001 启动、会话与登录门禁
- App 必须显示启动加载状态，并在启动失败时显示错误和重试入口。
- 未登录用户必须进入登录页。
- 已登录但未完成首评的用户必须进入首评页。
- 已登录且完成首评的用户必须进入首页。
- 登录页必须提供微信、Apple、手机号验证码和邮箱入口，并在提交前校验协议勾选。
- 登录失败、平台不可用、网络不可用或后端返回错误时，必须展示错误提示。

### FR-002 首评与学习路线
- 首评必须包含目标方向、表达卡点、当前输出水平和每日分钟数四步。
- 缺少目标方向、表达卡点或当前输出水平时，用户不能完成对应步骤。
- 选择英语面试必须写入 `job_interview`。
- 选择入职介绍必须写入 `onboarding_introduction`。
- 选择工作沟通必须映射到 `onboarding_introduction`，不得生成独立工作沟通场景。
- 选择日常服务不得写入真实官方场景，也不得出现在当前已实现场景范围中。
- 每日分钟数只作为偏好保存，不作为推荐队列长度控制需求。

### FR-003 首页与官方场景目录
- 首页底部必须只有情景学习、推荐表达、我的三个入口。
- 情景学习页必须加载官方场景目录。
- 当前官方场景目录只包括英语面试和入职介绍。
- 用户必须能查看场景标题、简介、标签、目标等级、表达数量和进度。
- 用户必须能搜索、分类筛选、加入、移除、设为当前场景并切换目标等级。
- 首页必须展示当前或已加入场景、掌握进度、到期复习、薄弱表达、个人素材、未完成会话和下一步建议。

### FR-004 听力热身
- 用户必须能从官方场景进入开始热身页。
- 热身页必须按当前场景和目标等级加载对话。
- 用户必须能播放、暂停、上一句、下一句和循环播放场景对话。
- 用户必须能切换听力模式和跟读模式。
- 跟读模式必须支持录制候选人台词、语音识别、完整度反馈和可用时的发音评分。
- 播放、录音、识别或评分失败时，页面必须展示可恢复错误。

### FR-005 推荐表达与每日队列
- 已加入场景的用户必须能在推荐表达页看到每日表达队列。
- 未加入场景的用户必须看到推荐表达空状态。
- 队列必须根据到期复习、薄弱表达和表达变体生成，并按优先级排序和去重。
- 推荐表达任务必须支持当前代码中的选择判断、听后复述、表达填空、意图回忆、接下一句、短语背诵、替换槽位、纠错复现、变体改写、流利挑战和跟读。
- 用户完成任务后，表达进度、最佳得分、转写、复习时间或掌握关联必须更新。

### FR-006 表达收藏
- 用户必须能收藏和取消收藏推荐表达。
- 收藏必须写入稳定 ID，重复收藏同一表达不得新增重复项。
- 收藏页必须展示有效收藏表达，并支持取消收藏。
- 收藏表达只保证复看，不要求自动生成复习任务。

### FR-007 语音场景模拟
- 用户必须能从官方场景和目标等级进入场景模拟。
- 场景模拟必须以语音作答为主路径，提供点击说话、录音、取消、提交和转写提交能力。
- 页面必须展示当前练习状态、目标进度和当前对话消息。
- 用户必须能请求提示，提示次数和内容必须可见。
- 用户必须能打开场景导航，查看目标等级、表达列表、当前节点、掌握、准备、到期复习、薄弱和练习统计。
- 用户必须能从当前等级场景导航跳转目标表达，或切换到其他目标等级。
- 同一用户再次进入同一场景同一等级时，符合条件的未完成会话必须自动恢复。

### FR-008 教练反馈与消息辅助
- 用户提交有效回答后，必须显示用户消息。
- 系统必须展示教练反馈、重试建议、表达建议、下一问题或可恢复错误中的至少一种。
- 命中目标表达时，必须能更新掌握反馈或掌握状态。
- 暴露问题时，必须能记录薄弱表达、错误模式或重试建议。
- 教练消息必须支持播放或播放失败提示。
- 教练消息必须支持中文翻译或翻译失败提示。
- 用户语音消息必须支持播放；录音文件失效时必须提示错误。
- 发音和语法评分可用时必须能展示反馈，不可用时不得阻塞主流程。

### FR-009 复盘、复习与个人 Wiki
- 一轮场景练习结束后，必须展示练习总结。
- 总结必须展示本轮学会、总掌握进度、遗忘曲线、薄弱标签或下轮重点。
- 总结后必须清理已完成的活跃会话。
- 系统必须把掌握表达、复习时间、薄弱表达、错误模式、个人素材或下轮目标写入本地状态中的至少一种。
- 写入结果必须能影响后续首页、推荐表达队列、个人 Wiki 或个人中心。
- 用户必须能在总结后开始下一轮练习。

### FR-010 我的、学习结果与设置
- 我的页必须展示资料、会员状态、练习数、连续天数和收藏数。
- 概览页签必须展示学习统计空状态或统计数据、收藏摘要、技能分布和学习资产入口。
- 历史页签必须展示学习历史空状态或练习记录，并支持查看详情和删除记录。
- 设置页签必须提供编辑资料、会员、收藏、提醒、主题、隐私政策、服务条款、注销和退出登录入口。
- 每日提醒开关和时间必须保存到本地通知设置。
- 主题切换必须更新并保存应用主题。
- 退出登录或注销后必须清理当前会话相关状态并回到登录门禁。

### FR-011 会员、学习报告与占位页
- 会员页必须展示当前计划、订阅方案、权益文案、购买按钮和恢复购买按钮。
- iOS/macOS 必须通过 Apple IAP 前端服务发起购买、恢复购买和订阅状态检查。
- Android 必须明确提示订阅未接入。
- 学习报告页必须调用学习画像接口；无数据时显示空状态。
- 离线内容页和成就页只能展示空状态或占位说明。

## 成功标准
- SC-001 未登录用户启动后看到登录页。
- SC-002 已登录未首评用户启动后看到首评页。
- SC-003 已登录已首评用户启动后看到首页三个主入口。
- SC-004 首评缺少目标方向、表达卡点或输出水平时不能继续完成。
- SC-005 英语面试或入职介绍能写入学习路线并在首页体现。
- SC-006 官方场景目录只按英语面试和入职介绍验收。
- SC-007 加入、移除、设为当前和切换等级会影响首页或后续练习入口。
- SC-008 听力热身可播放对话、切换句子、循环播放和录音跟读。
- SC-009 推荐表达队列能基于复习、薄弱和变体生成任务。
- SC-010 收藏表达后收藏页可见，取消后不可见，重复收藏不生成重复项。
- SC-011 场景模拟可语音提交回答，并展示反馈、提示、下一问题或错误。
- SC-012 未完成会话能在同场景同等级恢复。
- SC-013 练习结束后显示总结，并影响后续首页、推荐表达、个人 Wiki 或个人中心。
- SC-014 个人中心能查看学习结果入口、设置提醒、切换主题、管理会员入口和账号操作。
- SC-015 离线内容、成就、Android 支付、完整权益 gating 不被验收为已完成业务。

## 非目标
- 不新增日常服务、旅行、购物、医疗、校园等真实场景资产。
- 不把工作沟通写成独立官方场景。
- 不接入旧表达卡片课程或通用虚拟好友/自定义场景为当前主流程。
- 不把文本输入作为场景模拟主路径。
- 不要求收藏表达自动生成复习任务。
- 不要求薄弱表达跨全新场景自动复现。
- 不要求离线内容包下载、成就解锁、完整会员权益 gating 或 Android 订阅购买闭环。
- 不要求帮助反馈和联系我们形成客服工单闭环。
- 不要求后台 CMS、课程市场、社区、排行榜或真人导师。
- 不承诺后端 AI、ASR、TTS、评分、统计或学习报告服务在所有环境可用。
