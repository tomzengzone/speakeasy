# 当前 MVP 范围

## MVP 目标
交付一版与当前 Flutter 前端一致的官方职场场景表达练习闭环：学习者完成登录门禁和首评后，能从首页进入英语面试或入职介绍场景，完成听力热身、推荐表达练习、语音场景模拟、教练反馈、表达收藏、复习沉淀和练习总结，并在首页、推荐表达、收藏页或个人中心看到学习结果。

## 当前验收范围
- 启动与门禁：App 启动时有加载状态和失败错误页；未登录进入登录页，已登录未首评进入首评页，已登录且已首评进入首页。
- 登录页：提供微信、Apple、手机号验证码、邮箱登录/注册入口，所有登录提交前必须同意服务条款和隐私政策；具体登录成功依赖后端、平台能力和配置。
- 首评：用户选择目标场景、表达卡点、当前输出水平和每日分钟数；缺少关键选择时不能进入下一步或完成首评。
- 首评场景写入：英语面试写入 `job_interview`，入职介绍写入 `onboarding_introduction`，工作沟通映射到 `onboarding_introduction`，日常服务不写入官方场景选择。
- 首页主导航：首页底部入口为情景学习、推荐表达、我的。
- 官方场景资产：当前只按英语面试和入职介绍两个真实场景验收。
- 场景管理：用户可以搜索、按分类筛选、加入、移除、设为当前场景，并切换 L1/L2/L3 对应目标等级。
- 首页学习状态：首页展示当前或已加入场景、目标等级、表达进度、到期复习、薄弱表达、个人素材、未完成会话和下一步建议。
- 听力热身：用户可以播放/暂停场景对话、切换上一句/下一句、开启/关闭循环、切换跟读模式、录音跟读并看到完整度或发音相关反馈。
- 推荐表达：已加入场景后，用户可以看到每日表达队列；队列来自到期复习、薄弱表达和表达变体。
- 表达小练：推荐表达支持选择题、听后复述、表达填空、意图回忆、接下句、短语背诵、替换槽位、纠错复现、变体改写、流利挑战和跟读等当前代码已支持任务。
- 收藏：用户可以收藏、取消收藏表达，并在收藏页看到有效收藏；重复收藏同一表达不会新增重复项。
- 场景模拟：用户按官方场景和目标等级进入语音模拟，查看当前场景、目标等级、目标进度和场景导航；用户可请求提示、语音作答、取消录音、提交录音和播放自己的语音。
- 教练反馈：提交回答后，页面展示教练反馈、重试建议、下一问题、表达建议或可恢复错误；用户可播放或翻译教练消息。
- 会话恢复：同一用户再次进入同一场景同一等级时，符合条件的未完成会话自动恢复。
- 学习沉淀：练习过程和复盘会写入掌握表达、复习时间、薄弱表达、错误模式、个人素材、表达练习进度和场景进度。
- 练习总结：一轮练习结束后，用户看到本轮掌握、总进度、遗忘曲线、薄弱标签和下轮重点，并可开始下一轮。
- 个人中心：用户可以查看资料、学习概览、收藏摘要、学习历史、学习报告入口、已完成场景入口、提醒设置、主题切换、会员入口、隐私条款、服务条款、注销和退出登录。
- 会员页：展示会员方案、权益文案、购买/恢复购买入口和当前计划状态；仅承认 iOS/macOS Apple IAP 前端接入和 Android 未接入提示。
- 占位页：离线内容和成就只验收空状态或占位说明。

## 当前不作为 MVP 需求
- 日常服务、旅行、购物、医疗、校园等完整可练官方场景。
- “工作沟通”独立官方场景资产。
- 首评每日分钟数实际限制队列长度或推荐数量。
- 文本输入作为场景模拟主路径。
- 旧表达卡片课程 `LearningPage`/`LessonDetailPage` 的可达学习入口。
- 通用虚拟好友或自定义场景 `features/scenario/ScenePage` 的可达产品入口。
- 收藏表达后自动生成复习任务；收藏只保证收藏页复看。
- 薄弱表达跨全新场景自动复现。
- 离线包下载、成就解锁、完整会员权益 gating、Android 订阅购买闭环。
- 帮助反馈和联系我们的完整工单或客服闭环。
- 后台 CMS、课程市场、社区、排行榜、真人导师。
- AI、ASR、TTS、评分、统计或学习报告服务在所有环境中的后端可用性。

## MVP 成功定义
- 学习者可以通过登录门禁进入首评和首页。
- 学习者可以把英语面试或入职介绍写入学习路线，并在首页继续看到当前场景和目标等级。
- 学习者可以加入官方场景，完成一次听力热身或推荐表达练习。
- 学习者可以进入语音场景模拟，提交回答并看到教练反馈、下一问题、重试建议或可恢复错误。
- 学习者可以完成一轮练习并看到总结。
- 收藏、掌握、薄弱、复习或练习历史中至少一种结果能在后续首页、推荐表达、收藏页或个人中心体现。
- 每条 MVP 需求都能追溯到当前前端代码、资产、存储、服务调用或测试。
